# Das innere Produkt von Macdonald

Das innere Produkt von dem $j$-Vektor $A$ mit dem $k$-Vektor $B$ ist als
$$A\cdot B = \langle AB\rangle_{k-j}$$
erklärt.

Wenn $j > k$ ist, dann ist $k - j < 0$,  sodass $A\cdot B = 0$ ist.
