# Das innere Produkt von Hestenes

Für ein $r$-Blade $A_r$ und ein $s$-Blade $B_s$ ist das Skalarprodukt als
$$A_r\cdot B_s = \langle A_rB_s\rangle_{|r-s|}$$
erklärt.
