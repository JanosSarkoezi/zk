# Summe von Punkten

Für die Summe von Punkten im Raum wird ein Punkt mit zwei Daten karakterisiert
- Ein Zahlenfaktor $\mathfrak{m}$, genannt die Masse des Punktes
- Die Lage $f$ des Punktes im Raum

Somit kann ein Punkt $e$ als
$$e = \mathfrak{m}f$$
geschrieben werden.
