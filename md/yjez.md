# Linearkombination

Eine Linearkombination von Vetoren $v_1, ..., v_n$ aus einem rellen
Vektorraum $V$ ist ein Ausdruck der Form

$$
\alpha_1 v_1 + \cdots + \alpha_n v_n
$$

wobei die $\alpha_i \in \R$ für alle $i = 1, ..., n$.
