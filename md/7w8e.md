# Vim Extrahieren von Zahlen

Angenommen ich habe
```java
 public final static int PVAID_STATUS_OK = 158;
public final static int PVAID_VERANTWORTLICHER = 161;
public final static int PVAID_PRODUKTBEZEICHNUNG_INTERN = 164;
public final static int PVAID_ERWARTETE_MENGE_DER_WARE = 167;
public final static int PVAID_EINKAEUFER = 170;
public final static int PVAID_BESTELLER = 173;
public final static int PVAID_QUANTITAET = 176;
public final static int PVAID_BESCHAEDIGUNGEN = 179;
public final static int PVAID_MENGE_IN_ORDNUNG = 182;
public final static int PVAID_MENGE_NICHT_IN_ORDNUNG = 185;
public final static int PVAID_QUALITAET = 188;
public final static int PVAID_LIEFERORT_2 = 191;
public final static int PVAID_BISHER_GELIEFERTE_MENGE = 194;
```
und moechte mit vim nur die Zahlen behalten. Wie kann ich das mit norm machen?

Das ist ein klassischer Anwendungsfall für Vims **Normal-Mode-Befehle** in
Verbindung mit dem Befehl `:norm` (oder `:normal`).

Da deine Zahlen am Ende der Zeilen stehen, kannst du sie isolieren, indem du
alles löschst, was davor steht. Hier ist der effizienteste Weg, dies zu tun:

### Die Lösung mit `:norm`

Gib im Command-Mode (nach Drücken von `:` ) Folgendes ein:

```vim
:%norm 0df d$

```

**Was genau passiert hier?**

* `%`: Wendet den Befehl auf alle Zeilen an.
* `norm`: Führt die folgenden Befehle im Normal-Mode aus.
* `0`: Springt zum Anfang der Zeile.
* `df `: (**d**elete **f**ind Leerzeichen) Löscht alles bis zum ersten Leerzeichen (das Wort `public`).
* `     `: (Ein Leerzeichen nach dem `f`).
* `.`: Der Punkt wiederholt den Löschvorgang. Da deine Zeile mehrere Wörter hat, müsstest du `df ` eigentlich mehrfach tippen.

---

### Ein noch einfacherer Weg (Rückwärts löschen)

Oft ist es einfacher, von vorne bis zur Zahl zu löschen. Da alle deine Zeilen
mit einer Zahl enden und davor ein Leerzeichen oder Gleichheitszeichen steht,
kannst du auch dies nutzen:

```vim
:%norm d/ \d\+$

```

* `d`: Löschen.
* `/ \d\+$`: Bis zum Muster "Leerzeichen gefolgt von Zahlen am Zeilenende".

### Der "Vim-Profi" Weg (Suchen & Ersetzen)

Auch wenn du nach `norm` gefragt hast, ist Regex in Vim für diesen speziellen
Fall oft sauberer, da du auch das Semikolon am Ende loswerden willst:

```vim
:%s/.* \(\d\+\);/\1/

```

* Sucht nach allem bis zum letzten Leerzeichen, gruppiert die Ziffern
`\(\d\+\)` und ersetzt die ganze Zeile nur durch diese Gruppe `\1`.
