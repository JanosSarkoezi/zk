# Aufruf der sp_add_property

```sql
SELECT 'exec sp_add_persontag '
    + CONVERT(VARCHAR, PersonTagId)
    + COALESCE(', ''' + REPLACE(Value, '''','''''') + '''', ', NULL')
    + COALESCE(', ' + CONVERT(VARCHAR, ListelementAspect), ', NULL')
    + COALESCE(', ''' + REPLACE(Description, '''','''''') + '''', ', NULL')
    + ', ' + CONVERT(VARCHAR, HibernateVersion)
    + COALESCE(', ''' + LabelI18n + '''', ', NULL')
    + ', ' + CONVERT(VARCHAR, Valid)
FROM T_PersonTag
WHERE PersonTagId < 0
```
