title: Definition der Stammfunktion
type: def
tags: [analysis, integral]

---
Eine Funktion $F$ heißt **Stammfunktion** von $f$ auf einem Intervall $I$, wenn $F$ auf $I$ [[ableitung_definition|differenzierbar]] ist und für alle $x \in I$ gilt:
$$F'(x) = f(x)$$

--examples--
Heuser (Nr. 55): Ist $F_0$ eine Stammfunktion von $f$, so sind alle Funktionen der Form $F(x) = F_0(x) + C$ mit einer Konstanten $C \in \mathbb{R}$ ebenfalls Stammfunktionen von $f$. Dies folgt daraus, dass die [[ableitung_definition|Ableitung]] einer Konstanten null ist und Funktionen mit verschwindender [[ableitung_definition|Ableitung]] auf einem Intervall konstant sind (Folgerung aus dem Mittelwertsatz).
